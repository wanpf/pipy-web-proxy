# 将 http/https 代理日志写入 SQLite 数据库    
# 1. 启动 pipy-web-proxy 服务  
命令： pipy pjs/main.js
# 2. 浏览器设置 http/https 代理  
代理地址：代理服务器IP地址:9090  
代理服务端口为 9090  
# 3. 查看访问日志  
浏览器访问：http://代理服务器IP地址:6060
